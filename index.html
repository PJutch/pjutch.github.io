Calculingo

По сути клон Дуолинго но с матаном/дифурами

Поиск и создание коллекций заданий

Идеи доп. фич

Прикрутить нейронку для генерации решения (т. к. самому писать решения для всего долго)

Идеи доп. фич

Отслеживать стрики
(на уровне решил в день N задач из данной категории => получаешь +1 к стрику по данной категории) и сделать телеграм-бота,
спамящего уведомлениями
о сгорающем стрике

Плюс можно использовать для аутентификации
